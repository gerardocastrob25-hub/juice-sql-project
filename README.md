# juice-sql-project

Mialiana's juicery

## Overview

This project tracks juice sales for our small juice business.

The database stores:

- Products
- Product prices
- Sales quantities
- Sales dates

## SQL Skills Demonstrated

- CREATE TABLE
- INSERT INTO
- SELECT
- JOIN
- GROUP BY
- SUM
- AVG
- ORDER BY

## Business Questions Answered

- Total revenue
- Revenue by product
- Total bottles sold
- Sales by date
- Average daily sales
- Best-selling product
